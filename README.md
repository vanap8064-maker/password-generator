---

## 📝 Описание проекта
«Random Password Generator» — это десктопное приложение, созданное на языке Python. Оно позволяет пользователю быстро создавать надежные пароли, настраивать их длину и состав, а также просматривать историю генераций.

Приложение спроектировано с разделением на логические модули (интерфейс, генерация, хранение данных), что обеспечивает чистоту и читаемость кода и соответствует требованиям к итоговой аттестации.

---

## ✨ Основные функции
*   **Настройка длины:** Выбор длины пароля от 4 до 64 символов с помощью удобного ползунка.
*   **Выбор сложности:** Чекбоксы для включения букв (строчных и заглавных), цифр и специальных символов.
*   **Валидация:** Система проверяет ввод и выдает ошибку, если пользователь не выбрал ни одного параметра сложности.
*   **История паролей:** Последние сгенерированные пароли автоматически сохраняются в файл `history.json` и отображаются в таблице внутри приложения.
*   **Генерация:** Использование стандартной библиотеки `random` для обеспечения криптографической случайности.
